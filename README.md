# ***Bash_Password_Generator***
&nbsp;
## **DISCLAIMER:** 
#### Can Only be utilised fully, if the OS is Linux

## For Installation Setup:
&nbsp;
```
$ cd Sh_PasswordGen/Password_Gen/
$ chmod 744 setup.sh
$ ./setup.sh
```
## To run:
&nbsp;
### Simply write:
```
$ PasswordGenerator

OR

$ PasswordGenerator -h
```
